> **Um website elegante e moderno para um restaurante de culinária francesa, desenvolvido em WordPress.**

O **LeBonAppetit** é uma interface web focada em apresentar a experiência e a sofisticação de um restaurante francês. O site é gerenciável via painel do WordPress, oferecendo navegação fluida e adequação às normas de privacidade (LGPD).

## 🚀 Funcionalidades

* **Gestão de Conteúdo (CMS):** Totalmente integrado ao painel do WordPress para fácil atualização de textos e imagens.
* **Adequação LGPD:** Sistema integrado de permissões de cookies e política de privacidade.
* **Páginas Personalizadas:** Templates específicos para a página inicial e a seção "Quem Somos".
